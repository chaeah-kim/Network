# IP (Internet Protocol)

- 네트워크에서 정보를 송수신하기 위한 통신 규약을 의미함
- 인터넷을 사용하는 우리 모두에게 하나씩 주어진 주소 → IP주소라고 한다 (전 세계적으로 유일한 주소)
- TCP/IP 프로토콜을 사용하는 장비들을 구분해주기 위해 만든 것이 **IP address**
- IP address는 **네트워크 부분**과 **호스트 부분**으로 구성
- Network ID: 학교, 회사, 집 등 특정 네트워크 그룹을 식별 <br>
- Host ID: 해당 그룹 내에서 개별 컴퓨터나 스마트폰을 식별 <br>

- 동일 네트워크에서는 Network 부분은 모두 같고 Host 부분이 모두 달라야 한다.

→ IP주소를 네트워크 부분과 호스트 부분으로 구분해주는 역할을 하는 것이 **Subnet mask**

<br>
<br>

🟢**IPv4** 

- **32Bit**로 구성 → **8Bit씩** 나눠서 **4개의 Octet**으로 구분 (8bit.8bit.8bit.8bit) <br>
- 각 Octet을 10진수로 변환해서 표현한다 <br>
- 논리적 주소라고 부른다, 각 Octet의 **최소 값 0**부터 **최대 값 255**까지 가능 <br>
- 2^8은 256이지만 1이 아니라 0부터 사용하기 때문에 한 옥텟에 0~255까지 사용 <br>

**ex) 11000000.10101000.00001100.00000001**
**⇒ 192.168.12.1** 

(→ 십진수 한자리는 이진수 8자리로 만들어졌다. **이진수 8자리로 가장 큰수는 십진수로 255**) <br>

<br>

🟢 **Class** <br>

- IP 주소 범위에 따라 서브넷 마스크를 고정값으로 정한 것 <br>

**A class : 0 ~ 127**  <br>
→ 대규모 네트워크에 사용 (국가, 대기업) <br>
→ 127.0.0.1 루프백 주소로 사용 <br>
- 루프백 주소? <br>
    (네트워크 잘되는지 한바퀴 돌고 오는 용도로 사용, 네트워크 테스트 용도) <br>
- 10.x.x.x 사설ip

**B class : 128 ~ 191** <br>
→ 중규모 네트워크에 사용 (대학, 기관) <br>
- 172.16~31.x.x 사설ip <br>

**C class :192 ~ 223** <br> 
→ 소규모 네트워크에 사용 (가정, 소호) <br>
- 192.168.x.x 사설 ip  <br>

**D class : 224~ 239** <br>
→ 멀티캐스트 주소 <br>
<br>
**E class : 240 ~ 255** <br>
→ 군사목적용, 특수연구용으로 사용

⇒ 실제 쓸 수 있는 클래스 → **A,B,C** <br>
맨 앞자리 0 사용 불가능, 1부터 사용가능 (A class) <br>
공인 IP: 세계에서 유일한 IP <br>
사설 IP: 내부에서만 사용하는 것 , 회사, 집, 학교, 학원 등등 <br>
